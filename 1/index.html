Project 1

Single-scale Implementation

In this implementation, I align the parts by exhaustively searching over a window of possible displacements (from -15 to 15 pixels). For each possible displacement, I calculate a score using an image matching metric, and select the displacement with the best score. The image matching metric I use is Structural Similarity Index (SSIM) to evaluate the correct alignment between the images.
